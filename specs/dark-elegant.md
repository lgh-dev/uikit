# Dark Elegant UI 设计规范 v2

## 基本信息

名称: Dark Elegant v2（暗黑优雅 优化版）
版本: 2.2.0
适用场景: 设计工具、开发IDE、专业软件、高端产品
设计理念: 精致、神秘、高对比，在黑暗中绽放优雅

## 间距系统

基数: 4px（所有间距为 4 的倍数）

| 级别 | 尺寸 | 用途 |
|------|------|------|
| spacing-1 | 4px | 微间距、图标与文字间隙 |
| spacing-2 | 8px | 紧凑间距、列表项内部 |
| spacing-3 | 12px | 小间距、按钮内边距 |
| spacing-4 | 16px | 标准间距、卡片内边距 |
| spacing-5 | 20px | — |
| spacing-6 | 24px | 舒适间距、表单项间距 |
| spacing-8 | 32px | 大间距、卡片间距 |
| spacing-10 | 40px | — |
| spacing-12 | 48px | 板块间距 |
| spacing-16 | 64px | 大板块间距、页面分区 |


### 使用指南

- 组件内部：spacing-3 至 spacing-4
- 组件之间：spacing-6 至 spacing-8
- 板块之间：spacing-12 至 spacing-16

## 动画系统

### 时长

| 类型 | 时长 | 用途 |
|------|------|------|
| 快速 | 150ms | 微交互（按钮点击、开关切换） |
| 标准 | 250ms | 常规过渡（悬停、展开收起） |
| 缓慢 | 400ms | 页面切换、模态框出现 |


### 缓动曲线

| 类型 | 描述 | 用途 |
|------|------|------|
| 标准 | 先加速后减速 | 通用过渡 |
| 减速 | 快速开始缓慢结束 | 进入动画 |
| 加速 | 缓慢开始快速结束 | 退出动画 |

## 无障碍要求

### 颜色对比度

| 组合 | 对比度 | 标准 |
|------|--------|------|
| 主文字 / 卡片背景 | 17.4:1 | WCAG AAA |
| 次要文字 / 卡片背景 | 9.8:1 | WCAG AAA |
| 占位文字 / 输入框背景 | 4.7:1 | WCAG AA |


### 焦点指示

- 所有可交互元素必须有清晰的焦点状态
- 焦点环：2px 焦点边框 + 标准发光
- 禁止移除焦点指示而不提供替代方案


### 键盘导航

- Tab 键顺序符合视觉顺序
- Enter/Space 激活按钮和链接
- Escape 关闭模态框和下拉菜单
- 方向键导航列表和标签页

## 设计检查清单

色彩
- [ ] 页面背景使用页面背景色
- [ ] 层次分明，从黑到深灰逐级变化
- [ ] 白色作为主强调色
- [ ] 派生色用于交互状态

组件
- [ ] 每个交互元素都设计了悬停状态
- [ ] 每个交互元素都设计了聚焦状态
- [ ] 每个表单元素都设计了禁用状态
- [ ] 表单有错误和成功状态

间距
- [ ] 所有间距为 4px 的倍数
- [ ] 组件内外间距层次分明

动画
- [ ] 过渡时长合理（150-400ms）
- [ ] 入场用减速曲线，退出用加速曲线
- [ ] 禁用状态无动画

无障碍
- [ ] 对比度符合 WCAG AA
- [ ] 焦点环清晰可见
- [ ] 支持完整键盘操作

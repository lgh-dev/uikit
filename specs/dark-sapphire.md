# Dark Sapphire 暗黑蓝宝石

### 质感

- **玻璃拟态**：背景模糊 + 半透明叠加层 + 微发光边框
- **光晕效果**：蓝宝石色 glow 发光（box-shadow: 0 0 20px rgba(59, 130, 246, 0.25)）
- **冰晶高光**：顶部 1px 线性渐变高光，模拟玻璃折射

## 3. 交互

### 状态

#### 默认状态
- 透明度：100%
- 边框：1px solid border-glass
- 阴影：0 4px 20px rgba(0, 0, 0, 0.3)

#### Hover 状态
- 背景：rgba(59, 130, 246, 0.1)
- 边框：1px solid rgba(59, 130, 246, 0.4)
- 阴影：0 0 20px rgba(59, 130, 246, 0.2)

#### Active 状态
- 背景：rgba(59, 130, 246, 0.15)
- 边框：1px solid rgba(59, 130, 246, 0.5)
- 阴影：无
- 缩放：0.98

#### Focus 状态
- 边框：2px solid blue-500
- 阴影：0 0 24px rgba(59, 130, 246, 0.35)
- outline：none

#### Disabled 状态
- 透明度：0.4
- 背景：rgba(15, 23, 42, 0.6)
- 边框：1px solid border-glass
- 阴影：无

## 4. 动效

### 时长
| 名称 | 时长 | 用途 |
|------|------|------|
| fastest | 100ms | 开关、微交互 |
| fast | 150ms | hover状态 |
| normal | 200ms | 默认过渡 |
| slow | 300ms | 展开/收起 |
| slowest | 500ms | 页面切换 |

### 缓动曲线
| 类型 | 曲线 | 用途 |
|------|------|------|
| ease-out | cubic-bezier(0, 0, 0.2, 1) | 进入、悬浮 |
| ease-in | cubic-bezier(0.4, 0, 1, 1) | 离开、消失 |
| ease-in-out | cubic-bezier(0.4, 0, 0.2, 1) | 状态变化 |
| spring | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 弹跳效果 |

### 常见动效
- **按钮悬浮**：边框发光 + 蓝晕扩散
- **卡片进入**：向上滑入 + 渐显 + 玻璃模糊
- **加载动画**：旋转圆环（blue-500）
- **成功反馈**：对勾发光 + 脉冲
- **玻璃过渡**：background blur 渐变过渡

## 80/20 关键规则

1. **色彩**：#0a0f1c 深色背景 + #3b82f6 蓝宝石主色
2. **间距**：4px 基数，md (16px) 作为默认间距
3. **交互**：hover 蓝晕发光，focus 边框 + 光晕
4. **动效**：200ms ease-out 过渡
5. **圆角**：8px 按钮，12px 卡片，16px 弹窗
6. **阴影**：蓝宝石色 glow 效果
7. **玻璃拟态**：backdrop-filter blur + 半透明背景 + 冰晶高光
